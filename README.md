# Análisis de clientes y actividad de vuelos

Este proyecto consiste en un análisis exploratorio de datos sobre clientes de una aerolínea y su actividad dentro de un programa de fidelización.

El proyecto se ha desarrollado a partir de un caso práctico propuesto por Adalab durante mi formación en análisis de datos. A partir de los datasets proporcionados, se realiza un proceso completo de carga, limpieza, transformación, unión de tablas, análisis exploratorio y visualización de datos utilizando Python.

## Fuente del proyecto

Dataset y caso práctico proporcionados por Adalab.

El desarrollo del análisis, la limpieza de datos, las visualizaciones, la interpretación de resultados y la documentación han sido realizados por Carmen Gómez Zamora como parte de su portfolio profesional.
